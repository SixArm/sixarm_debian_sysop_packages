# SixArm.com » Debian » System operator packages

To add our repositority to your sources:

    edit /etc/apt/sources.list

Add this line:

    deb http://sixarm.com/deb/repository binary/

To install:

    apt-get update
    apt-get install sixarm-debian-sysop-packages
